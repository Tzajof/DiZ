Overview

DIZ_TEXTURE covers the loading and setup of OpenGL Textures.

Declaration

//This class is used to handle loading and properties of textures
class DIZ_TEXTURE {
public:
	//Declare the constructor and destructor
	DIZ_TEXTURE();
	~DIZ_TEXTURE();

	//Our functions- just the function to load the texture for now
	bool load();

	//Our texture properties- first just our texture handle
	GLuint tex;
	//The filename that we use when loading
	char *filename;
	//Filters- these specify the various filters to use for our texture, such as LINEAR or whatever else
	GLint minFilter, magFilter;
	//Mipmap indicates we want a mipmap texture, status indicates success of loading
	bool status, mipmap;
};

bool load()

This loads the texture. Should only be called once all the texture's settings have been set.

GLuint tex

Once loaded, this will be the texture's handle. Therefore when referencing the texture you must use this.

char *filename

This is the filename of the image you want to load. Currently, textures must be .bmp files.

GLint minFilter

This is the value that will be used for GL_TEXTURE_MIN_FILTER. Defaults to GL_LINEAR but other possible values include GL_NEAREST.

GLint magFilter

This is the value that will be used for GL_TEXTURE_MAG_FILER. Same defaults and possible values as minFilter.

bool status

Once loaded, a status value of true indicates the texture has been successfully loaded. False will indicate that an error occurred. This value starts off as false.

bool mipmap

This indicates whether or not we want to create a Mipmap texture.
