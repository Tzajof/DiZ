Overview

The dizPrint function is used to draw text to the screen using a specified font and message.

Declaration

//Function to draw a specified font to the screen
void dizPrint(DIZ_FONT *font, const char *fmt, ...);

DIZ_FONT *font

Pointer to the font that we want to use.

const char *fmt, ...

The message that you want to display. This may be followed by formatting arguments.

Example

The following example shows a common use of the dizPrint function. 
    //Declare our font
    DIZ_FONT font;

    //Set up your font here//

    //Position our text onscreen
    glRasterPos2f(0.0f, 0.0f);

    //Then print our message
	dizPrint(&font, "Hello World!");
