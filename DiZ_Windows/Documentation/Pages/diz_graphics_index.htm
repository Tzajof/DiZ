DiZ Documentation: DiZ Graphics

Index

Overview

DiZ Graphics makes up the entire graphics component of DiZ. It covers a range of simple graphics procedures, from window creation to texture handling and fonts.

To use DiZ Graphics, you must include diz-graphics.h in your project.
