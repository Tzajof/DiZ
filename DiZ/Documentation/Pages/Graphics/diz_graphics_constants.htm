Documentation: DiZ Graphics Constants

Index -> DiZ Graphics

Overview

This page contains a list of all constants declared in diz-graphics.h.

Constants

#define DIZ_2D 0

DIZ_2D is a constant used when specifying 2D graphics types for a DIZ_WINDOW.

#define DIZ_3D 1

DIZ_3D is a constant used when specifying 3D graphics types for a DIZ_WINDOW.

#define DIZ_FONT_BITMAP 2

DIZ_FONT_BITMAP is a constant used for specifying the Bitmap font type in a DIZ_FONT.

#define DIZ_FONT_OUTLINE 3

DIZ_FONT_OUTLINE is a constant used for specifying an Outline font type in a DIZ_FONT.
